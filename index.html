Making Your Web Pages Responsive

Viewport

A viewport is the visible area of a web page within a browser window. On mobile devices, the viewport varies depending on screen size and resolution. The viewport meta tag allows developers to control layout scaling.

Flexbox

Flexbox is a one-dimensional layout model used to arrange elements in rows or columns. It allows elements to grow, shrink, and align dynamically within a container, making it especially useful for responsive design.

Grid

Grid layout is a two-dimensional layout system that allows content to be organized into rows and columns with precise placement and alignment.
